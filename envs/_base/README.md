# ```_base``` Development environment
The scripts included in this folder will setup the developer's environment. This includes Kubernetes, cloud CLI tooling, and other tools required for building or interacting with the environment.

# Overview
This scripts in this folder are designed to build out a Kubernetes environment for development in different types of Kubernetes environments, in the cloud or local on Minikube. No matter which environment you decide to deploy, the client tools folder needs to be deployed to your workstation so that it can deploy and interact with the environment you look to deploy.
